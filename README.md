El archivo run_analysis.R debe archivarse en C:\Users\yourname\DAta\UCI HAR Dataset\

Ejecutar en RStudio: setwd("C:\\Users\\yourname\\Data\\UCI HAR Dataset\\"), seguido de  source("run_analysis.R")
